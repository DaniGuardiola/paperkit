Paperkit
Paperkit Quick start Components Demos Reference Paperkit is a Material Design web framework that works on top of native and standard technologies: HTML, CSS and JavaScript

Our main goal is to provide a simple and fast way to create webpages and apps following the latest guidelines from Google. We do not use any other framework or CSS preprocessor (like jQuery or Sass).

Getting ready Basic layout Tags and attributes
1- Import paperkit to your project
Bower
Minified (recommended) 
bower install materializer-min
Full size 
bower install materializer
Manual download
Minified (recommended) Full size
Note: extract on your project's folder 2- Include the css and js files
Important: include before every other css or js that overwrites Materializer styles and scripts
<head>
    <link rel="stylesheet" href="<path>/materializer/materializer.css">
    <script src="<path>/materializer/materializer.js"></script>
</head>
Note: <path> is the folder containing the Materializer folder. If you installed it with bower, the path will be 'bower_components'. Also, remember that the Materializer folder has '-min' at the end when it is the minified version. 3- Load and initialize Paperkit
Note: you can choose a different variable name other than 'paperkit'
var paperkit = new Materializer(); // Instantiate Paperkit on a variable of your choice
window.addEventListener('load', function(){ paperkit.init() }); // Initialize Paperkit on window load
 Next step: basic layout Top level elements
Childs of body tag, they define the basic layout.
They are <md-toolbar>, <md-sidemenu>, <md-fab> and <md-content>. 
<body>
	<md-toolbar></md-toolbar>
	<md-sidemenu></md-sidemenu>
	<md-fab></md-fab>
	<md-content></md-content>
</body>
 Content elements
Childs of md-content tag, they are the actual content of the page.
They can be any element, from <md-list> to <md-input>. 
<body>
	<md-toolbar></md-toolbar>
	<md-sidemenu></md-sidemenu>
	<md-fab></md-fab>
	<md-content>
		<md-list>
			<md-tile>
				<md-icon md-image="icon:android" md-fill="green"></md-icon>
				<md-text>I'm a list tile on the content tag!</md-text>
			</md-tile>
		</md-list>
	</md-content>
</body>
